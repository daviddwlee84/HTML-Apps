# HTML Apps

Single HTML as an app. Used to host random stuff generated by LLM.

## Getting Started

Start HTTP server

```bash
just
```

Create new app/page

```bash
# (recommend)
just new-app-interactive
# or
just new-app new_app_name
# or
mkdir apps/new_app_name && touch apps/new_app_name/index.html
```

Update index

```bash
just gen-index
```
